# Taller 2 - Flutter Firebase

Aplicación móvil desarrollada en Flutter con integración de Firebase.

## Funcionalidades implementadas

- Registro de usuarios con Firebase Authentication.
- Inicio de sesión con correo electrónico y contraseña.
- Almacenamiento de datos de usuarios en Firebase Realtime Database.
- Navegación hacia el catálogo de películas después de la autenticación.

## Tecnologías utilizadas

- Flutter.
- Dart.
- Firebase Authentication.
- Firebase Realtime Database.
